# resas-api-proxy
A simple proxy server for RESAS-API

## Project Setup
```sh
yarn install
```

### Compile and Hot-Reload for Development

```sh
yarn dev
```

### Type-Check, Compile and Minify for Production

```sh
yarn build
```

### Lint with [ESLint](https://eslint.org/)

```sh
yarn lint:fix
```

### Format with [Prettier](https://prettier.io/)

```sh
yarn format:fix
```
